# Project Statement: Math Quiz Generator CLI

## Problem Statement
The project addresses the lack of simple, interactive tools for students to practice basic arithmetic skills rapidly.The goal is to provide an engaging alternative to traditional learning methods, offering immediate feedback to reinforce learning effectively.

## Scope of the Project
The project is scoped as a Command Line Interface (CLI) application. It will focus on generating simple, random arithmetic questions (addition, subtraction, multiplication), validating user input, and providing a final score summary. The project does not include saving high scores or complex user profiles.

## Target Users
* Students (primarily level) needing practice in basic math.
* Any user looking for quick mental math practice.

## High-Level Features
1. Generation of randomized math questions.
2. Real-time answer validation.
3. Final score summary display.
